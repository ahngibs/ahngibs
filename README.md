# 👋 안녕하세요, 웹 개발자입니다

웹 서비스 개발 및 운영 경험이 있는 개발자입니다.
Classic ASP 기반 채용 솔루션의 유지보수 및 커스터마이징을 담당해 왔으며
서비스 운영 과정에서 발생하는 다양한 요구사항을 기능 개발과 데이터 처리로 해결해 왔습니다.

기존 **ASP / .NET 기반 관리자 시스템을 NestJS / React 구조로 전환하는 프로젝트에 참여하며**
Node.js 기반 API 구조와 프론트엔드 분리 구조를 경험했습니다.

현재는 **Node.js / NestJS 기반 백엔드 개발 역량을 확장하고 있습니다.**

## 📌 프로젝트

### ASP 관리자 시스템 (샘플 프로젝트)

Classic ASP 기반 관리자 페이지 예제 프로젝트

주요 기능

* 회원 목록 조회
* 검색 및 페이징
* 관리자 기능 구현
* MSSQL 데이터 처리

---

### NestJS API 구조 학습 프로젝트

기존 관리자 시스템의 **ASP / .NET 구조를 NestJS 기반 API 구조로 전환하는 과정에서 학습한 내용을 정리한 프로젝트**

주요 내용

* NestJS 기반 REST API 구조 이해
* API 기반 데이터 처리 방식 학습
* MSSQL 연동
* 관리자 기능 API 설계 경험

---

### GA4 Analytics API 연동

Google Analytics 데이터를 API로 조회하는 예제 프로젝트

주요 기능

* GA4 API 호출

* JSON 데이터 처리
